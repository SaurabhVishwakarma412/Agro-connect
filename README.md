# Agro-Connect 🌿

A full-stack web platform that connects **farmers, wholesalers, and consumers** for horticultural crops, enabling **direct trade** and **reducing middlemen**. The system promotes transparent pricing, real-time product listings, and better market accessibility.

---

## 🚀 Key Highlights

- 🌾 **Direct Crop Marketplace** connecting farmers, wholesalers, and consumers  
- 🔐 **User Authentication** with secure login and registration  
- 📦 **Dynamic Product Listings** for horticultural crops  
- 📊 **Dashboard Insights** for data-driven decisions  
- 💾 **PHP + MySQL Backend** for efficient data storage and processing

## ⚙️ Installation & Setup

```bash
# Clone the Repository
git clone https://github.com/your-username/Horti-Connect.git

# Navigate into the project
cd Horti-Connect
```

### Database Configuration

Import the `horticulture.sql` file into **MySQL**, then update database credentials in:

```
backend/config/db.php
```

```php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "horticulture";
```

### Run the Project

Start a local server using **XAMPP** or **WAMP** and open the project in your browser.
